# AI Ad Agency - Next.js Landing Page

A premium, cyber-futuristic landing page for an AI Ad Agency built with modern web technologies.

## Tech Stack
- **Framework:** Next.js 14 (App Router) + React 19
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Icons:** Lucide React

## Design System
- **Theme:** Pure Dark Mode (Obsidian Black #000000, #0a0a0a, #111111)
- **Typography:** Geometric sans-serif (Outfit/Inter)
- **Style:** Cyber-Futuristic Glassmorphism with neon accents

## Core Sections
- **Header:** Glassmorphic fixed navbar with glowing CTA button
- **Hero:** Terminal window showing 13-phase AI orchestration process
- **AI Creative Studio:** Pomelli integration with step-by-step grid
- **System Architecture:** 4-layer diagram (UI → Backend → MCP → APIs)
- **MCP Demo:** Video/code player with protocol details
- **Features:** Bento-box style cards
- **ROI Calculator:** Interactive savings calculator
- **Pricing:** Sleek tiers with highlighted Agency plan
- **Footer:** Minimalist with "Made with love by Adfordge AI team 💖"

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the project.
